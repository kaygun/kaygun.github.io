Path ideals

Description of the problem

Let G = (V,E) be a directed graph without loops and multiple edges. In other words V is a finite set and E is a subset of V × V which does not contain elements of the form (x,x) with x ∈ V. I will also assume that G contains no oriented cycles and V is a subset of the set of natural numbers ℕ of the form {0, 1, …, n} and that edges in G are increasing. Increasing labelling is possible in cycle-free graphs because of topological sort.

In this context a path is a sequence of vertices (v0,…,vn) such that any two consecutive vertices are connected by an edge. Today, I will develop an algorithm and the accompanying lisp code which will give the set of all paths which contain specific paths as subpaths.

Some poset theory

You don’t need to read this section if you are only interested in the algorithm but my warped mathematician mind requires that I think about it.

Basics and few examples

A partially ordered set (a poset in short) is a pair (P,⊢) where P is a set and ⊢ is a subset of the cartesian product P × P. Tradition dictates that instead of writing (x,y) ∈ ⊢, we write x ⊢ y. The subset ⊢ must satisfy certain conditions:

  • Reflexivity: for every x ∈ P we must have x ⊢ x
  • Anti-symmetricity: for every x, y ∈ P, if x ⊢ y and y ⊢ x then x = y
  • Transitivity: for every x, y, z ∈ P, if x ⊢ y and y ⊢ z then x ⊢ z

The set of natural numbers ℕ together with the ordinary order relation ≤ is an example of a poset. Another good example is the set of English words and the lexicographical ordering + reflexivity. But, let me give you a non-example: let P be the set B = {0, 1} and let ≺   := B × B = {(0,0), (0,1), (1,0), (1,1)} The set ≺ satisfies reflexivity and transitivity but doesn’t satisfy anti-symmetricity.

The relevant example

The relevant example for me is the set of all paths in a directed graph under the “subpath” relation. Let P(G) be the set of all paths in G and I will define  ⊢  ⊆ P(G) × P(G) as follows: Let α, β ∈ P(G). Assume α = (a0,a1,…,an) and β = (b0,…,bm). I will write α ⊢ β if α appears as a full segment of β. That is, if there exists an index i such that bi + j = aj for any j = 0, …, n. The fact that ⊢ is reflexive and transitive is simple to see. Showing that ⊢ is anti-symmetric requires that we consider the lengths of the paths involved: If α ⊢ β and β ⊢ α then the lengths of α and β are equal, and since α appers in β as a full segment the only way this can happen is when α = β.

Order ideals

An order ideal in a poset (P,⊢) is a subset I which satisfies the following condition:

For all x, y ∈ P if x ∈ I and x ⊢ y then y ∈ I

Here is another way of describing order ideals. For any x ∈ P and A ⊂ P we let U(x) := {y ∈ P ∥  x ⊢ y}  and  U(A) := ⋃x ∈ AU(x)  Then I ⊆ P is an order ideal if for every x ∈ I we also have U(x) ⊆ I.

Recasting the original problem

I can now recast the original problem: develop an algorithm which constructs U(A) the order ideal in (P(G),⊢) which contains a given subset A ⊆ P(G).

An algorithm

Today’s algorithm will use the algorithms Beginning-At and Ending-At from an earlier post. Respectively, the Beginning-At and Ending-At algorithms construct the set of all paths in G beginning and ending at a given vertex v ∈ V. Today’s algorithm in pseudo-code is as follows;

Function Order-Ideal
    Input: a set A of paths
    Output: the order ideal U(A) which contains A
    Initialize: F <- {}
Begin
    For each alpha in A do
        Let a <- source(alpha) and b <- target(alpha)
        For each beta in Beginning-At(b) do
            For each gamma in Ending-At(a) do
                Add the concatenation gamma*alpha*beta to F
            End for
        End for
    End for
    Return F
End function

(defun order-ideal (A G)
    (loop for alpha in A
          append (let ((a (car alpha))
                       (b (car (reverse alpha))))
                      (loop for beta in (beginning-at b G) 
                            append (loop for gamma in (ending-at a G) 
                                         collect (concat gamma (concat alpha beta)))))))
